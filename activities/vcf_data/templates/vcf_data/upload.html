Do you have whole genome or exome VCF data?

You're probably in the wrong place if you have data from 23andMe or AncestryDNA.

This upload form is only intended for whole genome or exome data in the VCF data format.

Data upload instructions

Please let us know at support@openhumans.org about any issues with the form. Thank you for bringing your data to Open Humans!

1. Find the correct genome file.

  • It should have a .vcf file suffix, or a compressed version of this (.vcf.gz or .vcf.bz2).
  • It should be fairly large. Probably over 100 MB, maybe over 1GB.
